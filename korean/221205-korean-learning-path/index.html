Korean Learning Path

Posted on 2022-12-05

关于韩语的自学路径，一些经验总结与整理。

学习路径

笔者陆续进行了日语和韩语的自学。参考个人进行日语学习的路径，把日语和韩语学习也分为以下几个阶段。

  1. 发音与字母
    在这个阶段，主要学习的是字母（日语假名/训名正音）的构建以及正确发音，做到见到能立刻读出。韩语这里的难点的连音，ㅗ/ㅓ分辨、ㅐ/ㅖ分辨、ㅒ/ㅖ分辨。
  2. 基本语句与单词
    在这个阶段，需要掌握一些最最基本的语句（问候/挨拶/인사）和其中的语法，同时掌握几十个最最基本的词汇（가다/오다）
  3. 语法系统构建
    在这个阶段，要构建基本的语法框架和系统
    • 词法：词性及其活用形
      包括动词（重点）与形容词的各种活用变形：时态、语态、敬语、疑问、否定、意志（使役）等等
    • 句法：句子如何构建
      日语和韩语的句子“主语+宾语+谓语”的结构，在连接和指示每个组成部分的时候需要用到特定助词，用来提示成分，每个助词前都需要搭配特定的活用形。此外表示特定的语气和意志有特定的固定句型（例如尊敬语）。
      比起词法，句法更加浩如烟海，而词法更为重要和基础。优先掌握常见句法，其他可以慢慢积累。学习每一个句型的时候都一定要搭配两到三个例句。
  4. 语料输入
    在这个阶段需要大量的语料输入，包括书面文字和视听材料，通过视觉和听觉两种刺激，一方面不断积累新的词汇和句型，另一方面积累文化素材，去理解语言背后的文化意义（例如平语/敬语）。同时不能死读课本，也要在网络上学习流行语和热门事件。这个阶段的关键是善用搜索和辞典。
  5. 语言输出
    语言能力可以概括为“听说读写”四大方面，通过词汇和语法的学习，以及语料的输入可以积累语感，但是要真正化为己用，需要输出。写作的输出包括翻译训练和日记作文的习作，口语的输出则需要通过对话（或者自言自语）
  6. 长期学习
    语言学习不是一蹴而就的，只有通过长期的积累才能真正掌握语言。

相较于传统课本（标日/延世韩国语）来说，这是一种集中度更高的方法，更适合这两种近似的语言中已经掌握其中一种的学习者。

词法·述语与语干

日语与韩语的活用/变形有这样的不同：

日语 韩语
区分形容词和动词
动词分为一段/五段/～する
形容词分为い段/ウ段/形容動詞(～だ)
各自不同活用变形方式
统称用言*		 不区分形容词和动词
统称述语（술어）
(全部以-다结尾)
遵循同一种活用/变形

与用言相对应的、不会发生活用/变形的词汇，在日语中被称作体言

将述语中中不会变化的部分称为语干，变化的部分称为词尾。在韩语的述语中，语干是除了 다 以外的部分。

韩语述语活用/变形有三个主要要素：

  1. 语干最后有无收音(받침/バッチム)
  2. 语干最后是阴性母音(ㅓㅕㅜㅠㅡ)还是阳性母音(ㅏㅑㅗㅛ)
  3. 语干最后的收音是否为ㄹ

关于时态、命令、原因的活用表可以参考韓国語動詞の活用表

句法

1. 基本结构与助词

在这篇文章， 笔者整理了基础的结构和句型，以及一些常见的助词。根据日语的类比，可以大致掌握一些类似用法。但是韩语的变形在涉及助词连接和活用时候相较于日语更加复杂，看似只能记忆，但是实际可以从语基的角度寻找其中规律

2. 三种语基

关于语基的内容，这篇知乎专栏有非常详尽的讲解。总结起来，各种各样述语的活用可以被概括成三种语基：

  • 第I语基：보다 → 보-，먹다 → 먹-。
  • 第II语基：보다 → 보-，먹다 → 먹으-。
  • 第III语基：보다 → 보아-，먹다 → 먹어-。若词干以元音结尾，则「아」或「어」常常与词干的末字缩合，例如보아-常常进一步缩合成봐-。

这是规则的语基结构，此外还有不规则(变格)语基，在文中 4.1.6 词干以辅音结尾的不规则用言总结 和 4.2 词干以元音结尾的不规则用言 有讲述。

通过语基的整理，我们把连接述语和助词的方法汇总并且系统化，除了小部分特殊用法，大部分的语法规则也就已经掌握规则、可以举一反三了。
